# test-vscode-extensions

A simple repository to test and publish Visual Studio Code extensions, using different package managers.

> [!NOTE]
> These extensions are only for testing purposes, and does not provide any functionality.

Currently the following packages are in use:

- [npm](https://www.npmjs.com/)
- [yarn](https://yarnpkg.com/)
- [pnpm](https://pnpm.io/)
